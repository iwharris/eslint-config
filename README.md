# eslint-config
Personal eslint config

## Installation and usage

```bash
npm install --save-dev @iwharris/eslint-config
```

Use the following statement in `.eslintrc`:

```json
{
    "extends": "@iwharris/eslint-config"
}
```

Or, in `package.json`:

```json
"eslintConfig": {
    "extends": "@iwharris/eslint-config"
}
```
